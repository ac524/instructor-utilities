Click the different parts of the button's to see different results in the console

When you click the red part of the button (The part inside the <span> tag) you will see `event.target` log the <span> instead of the <button>. In contrast, the line of code using `this` will always log the element matching the `.event-btn` selector in the event listener configuration.

Code

    
    $("#btn-container").on("click", ".event-btn", function(event) {

        // `event.target` will refer to the clicked element, so if the child  is clicked, then `event.target` will be the span.
        console.log("event.target = ", event.target);

        // `this` will always refer to the element matching the sub selector in our "on click" definition. So for this code, it will be the element that matches the ".event-btn" selector.
        console.log("this = ", this);

    })
